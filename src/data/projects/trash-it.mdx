# Trash It: Recycling with Computer Vision

## Overview

Trash It is a full-stack, containerized computer vision app that analyzes photos to identify recyclable objects, helping users sort waste correctly. It combines OpenCV with Azure Computer Vision APIs and runs containerized on Azure.

**🏆 Winner — HackHive 2024** (Ontario Tech University's inaugural hackathon).

## Features

- Photo analysis that identifies recyclable objects
- OpenCV + Azure Computer Vision classification pipeline
- Django backend, containerized with Docker
- Deployed live on Azure App Service
