# Elva: AI Aid for Alzheimer's & Dementia

## Overview

Elva is an AI-powered web application that assists individuals with Alzheimer's and dementia. Using computer vision, it recognizes familiar faces and identifies everyday objects, helping patients stay oriented and connected to the people around them.

**🥉 3rd Place Winner — HackHive 2025** (Ontario Tech University's annual hackathon).

## Features

- Face recognition to identify family members and caregivers
- Everyday object identification using computer vision
- Simple, accessible interface designed for patients
- Built with Python, Flask, and OpenCV

## Impact

Elva placed 3rd out of all teams at HackHive 2025, recognized for tackling a meaningful real-world healthcare problem with practical AI.
